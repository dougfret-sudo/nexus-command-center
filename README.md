# Yellow Iron Command Center (v1.0.0)
**Visual Intelligence Dashboard for Heavy Equipment Arbitrage**

## 🏗️ Overview
This dashboard serves as the UI layer for the Nexus ecosystem. It provides real-time visualizations of equipment price trends, hour-to-value analysis, and export margin calculations for global arbitrage (Midwest to Dubai).

### ⚙️ System Logic
- **[ROI Analyzer]**: Interactive calculator for "Landed Costs" including inland freight and port fees.
- **[Asset Mapping]**: Visual heatmaps of high-value private listings in the Michigan region.
- **[Particulars Filter]**: Advanced filtering for serial prefixes and machine hours to identify prime export units.

## 🛠️ Technical Stack
- **Frontend**: Streamlit (Python-only UI)
- **Data Link**: SQLite3 / Nexus API
- **Charts**: Plotly / Altair
